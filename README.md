# DnD
Gioco D&amp;D fatto in Java da JAVITA157

Siamo riusciti grazie al grande contributo di Manuel a creare personaggi, classi, competenze di ogni personaggio e il tiro dei dadi.
Dovremmo organizzarci per poter attribuire numero massimo di punti competenza da poter attribuire, dividere il altre classi.
Primi livelli di storia giocabile con scelte, aggiornare la creazione personaggio implementando con punti abilità, iniziativa ed equipaggiamento. Creare i primi oggetti come armi, armature, pozioni ecc...
